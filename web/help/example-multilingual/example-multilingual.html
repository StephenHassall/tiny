Example: Multilingual

Overview

In this example we are going to show what it would be like to have a single controller JavaScript file that can have multiple HTML files, one for each different language, and be able to switch between them. We will have 2 controllers, one for an About area and the other of a SwitchLanguage area that will allow the user to change the current language.

About Controller

The About controller will have 3 HTML files for English, Spanish and French. The About controller will load the required HTML file when it is created depending on which langauge is the current one.

4  <p>
5    This example shows you how you can use different HTML files dynamically.
6    In this case we are using 3 different languages and switching between them.
7  </p>
8</div>

When the About controller is created it checks to see which langauge is the current one. It then selects which HTML file to use when loading up. The HTML will be different but the CSS file will aways be the same, in this example.

1export default class AboutController extends Controller {
2  constructor() {
3    // Set the language dependant HTML
4    let htmlPage = 'about-en.html';
5    if (window.localStorage.langaugeCode === 'es') htmlPage = 'about-es.html';
6    if (window.localStorage.langaugeCode === 'fr') htmlPage = 'about-fr.html';
7
8    // Call the super with the page to load
9    super({
10      htmlPath: htmlPath,
11      cssPath: 'about.css',
12      importMetaUrl: import.meta.url,
13      useShadowDom: true
14    });
15  }
16}
17
18// Define controller web component
19customElements.define('about-controller', AboutController);

Switch Language Controller

The Switch Language controller will again have 3 HTML files for English, Spanish and French. Just like the About controller, it will load the required HTML depending on what the current langauge is.

1<!-- English: switchLanguage-en.html -->
2<div class="switchLanguage">
3  <p>Select the language to use.</p>
4  <button onclick="getController(this).setLanguage('en')">English</button>
5  <button onclick="getController(this).setLanguage('es')">Spanish</button>
6  <button onclick="getController(this).setLanguage('fr')">French</button>
7</div>

1<!-- Spanish: switchLanguage-es.html -->
2<div class="switchLanguage">
3  <p>Seleccione el idioma a utilizar.</p>
4  <button onclick="getController(this).setLanguage('en')">Inglesa</button>
5  <button onclick="getController(this).setLanguage('es')">Española</button>
6  <button onclick="getController(this).setLanguage('fr')">Francesa</button>
7</div>

1<!-- French: switchLanguage-fr.html -->
2<div class="switchLanguage">
3  <p>Sélectionnez la langue à utiliser.</p>
4  <button onclick="getController(this).setLanguage('en')">Anglaise</button>
5  <button onclick="getController(this).setLanguage('es')">Espanol</button>
6  <button onclick="getController(this).setLanguage('fr')">Français</button>
7</div>

Each HTML is using the same controller JavaScript code. This means they will use the same controller functions. The setLanguage function takes the same set of 3 parameters, en, es or fr, no matter which language is being shown.

1export default class SwitchLanguageController extends Controller {
2  constructor() {
3    // Set the language dependant HTML
4    let htmlPage = 'switch-language-en.html';
5    if (window.localStorage.langaugeCode === 'es') htmlPage = 'switch-language-es.html';
6    if (window.localStorage.langaugeCode === 'fr') htmlPage = 'switch-language-fr.html';
7
8    // Call the super with the page to load
9    super({
10      htmlPath: htmlPath,
11      cssPath: 'switch-language.css',
12      importMetaUrl: import.meta.url,
13      useShadowDom: true
14    });
15  }
16
17  setLanguage(languageCode) {
18    // Store the locale language code to use
19    window.localStorage.langaugeCode = languageCode;
20
21    // Reload the current page
22    location.reload();
23  }
24}
25
26// Define controller web component
27customElements.define('switch-language-controller', SwitchLanguageController);

Putting it all together

All we need to do now is put the About controller and the Switch Language controller together on the same page.

1<!DOCTYPE html>
2<html lang="en">
3<header>
4  <title>Tiny Web Application Framework - Example: Multilingual</title>
5  <script type="module" src="about.js"></script>
6  <script type="module" src="switch-language.js"></script>
7</header>
8<body>
9  <about-controller></about-controller>
10  <switch-language-controller></switch-language-controller>
11</body>
12</html>

First we need to include the controller's JavaScript files. You need to include the type="module" attribute when loading web components. Then you need to add the controller's tags. These are the <about-controller> parts. You can see the final result here.
